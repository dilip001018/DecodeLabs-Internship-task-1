# Data Cleaning and Preprocessing Using Excel

## Project Overview

This project focuses on cleaning and preprocessing raw data using Microsoft Excel. The dataset was cleaned by handling missing values, removing duplicate records, correcting inconsistencies, and applying Excel formulas to improve data quality and prepare it for analysis.

---

## Objectives

* Identify and handle missing values
* Remove duplicate records
* Standardize data formats
* Clean inconsistent entries
* Apply Excel formulas for data processing
* Prepare data for analysis and reporting

---

## Tools Used

* Microsoft Excel
* Excel Formulas and Functions

---

## Excel Features and Formulas Used

* Remove Duplicates
* Filter and Sort
* Conditional Formatting

* Data Validation

## Data Cleaning Steps Performed

### 1. Handling Missing Values

* Identified blank cells
* Filled or removed missing data where necessary

### 2. Removing Duplicates

* Used Excel's Remove Duplicates feature
* Eliminated redundant records

### 3. Data Formatting

* Standardized date, text, and numeric formats
* Corrected inconsistent values

### 4. Data Validation

* Checked data accuracy and consistency
* Applied validation rules where required

### 5. Formula-Based Processing

* Used Excel formulas to clean, transform, and analyze data

## Results

* Improved overall data quality
* Removed duplicate and inconsistent records
* Standardized dataset structure
* Generated a clean dataset ready for analysis
